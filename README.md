# SEO Title Optimizer Chrome Extension

A Micro-SaaS Chrome Extension to supercharge your on-page SEO. Analyze your current webpage and get instant, SEO-friendly title suggestions. Perfect for bloggers, marketers, and small business owners.

**Features:**
- On-page title, H1, and paragraph analysis.
- 3-5 AI-powered title suggestions.
- Character count and keyword density metrics.
- One-click copy for easy implementation.

Built with Manifest V3 for future-proof performance.
